=head1 NAME

Test::Kwalitee - Test how good your software is

=head1 SYNOPSIS

=head1 DESCRIPTION

Kwalitee is an automatically-measurable gauge of how good your software is.
That's very different from quality, which a computer really can't measure in
a general sense.

Using this module you can identify potential problems or shortcommings in
your distribution.
